# /clarify-roles
Make roles and ways of working explicit across functions.
## Steps
1. **Assign roles** — Build the RACI for key decisions using `raci-matrix` skill.
2. **Agree norms** — Co-create how the functions work together using `working-agreement` skill.
3. **Anchor to outcomes** — Confirm what success means using `okr-alignment` skill.
4. **Capture it** — Record the agreements using `decision-log` skill.
## Output
A role-clarity pack: a RACI, a working agreement, and a decision log seeded with the founding decisions.
Consider following up with `/set-up-intake` if request handling is still ad hoc.
